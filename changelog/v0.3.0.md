# 0.3.0 - 2026-03-14

This is an important release for this crate:

1. It now exists in its own repository: <https://github.com/serenity-rs/voice-model>
2. Support for [Discord's audio/video end-to-end encryption][dave], or DAVE for short, has been added.

In 2026, DAVE has become mandatory for audio/video calls. Without it, voice connections get rejected and no audio may be sent nor received. This release adds the necessary types and logic to serialise and deserialise DAVE protocol messages.
